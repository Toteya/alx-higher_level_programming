`MyList` class, inherits `list`
--------------------------------------------------------------

import the MyList from 1-my_list:
	>>> MyList = __import__('1-my_list').MyList

	>>> my_list = MyList()
	>>> my_list.append(3)
	>>> my_list.append(1)
	>>> my_list.append(2)
	>>> my_list.append(5)
	>>> my_list.append(4)

	>>> my_list.print_sorted()
